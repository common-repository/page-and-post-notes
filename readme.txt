Keep notes for each page and post within it. 

== Description ==

Save important information or data linked to page or post. Save it within page & post so that you can find it later quickly.


== Installation ==

* Upload the Page and Post Notes plugin to the `/wp-content/plugins/` directory
* Activate the Page & Post Notes plugin through the 'Plugins' menu in WordPress


== Frequently Asked Questions ==

= How to use Page and Post Notes plugin? =
* After plugin activation, it creates a small icon in top right corner of each page and post on editing it. 
* Notes can be entered there for a particular page or post.
